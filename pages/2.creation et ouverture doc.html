Introduction

Les fichiers Word sont appelés des documents. Chaque fois que vous démarrez un nouveau projet dans Word, vous devrez créer un nouveau document, qui peut être vierge ou basé sur un modèle. Vous devrez également savoir comment ouvrir un document existant.

Pour créer un nouveau document vierge :

Lorsque vous commencez un nouveau projet dans Word, vous voudrez souvent commencer avec un nouveau document vierge.

 

1.     Sélectionnez l'onglet Fichier pour accéder à la vue Backstage.

2.     Sélectionnez Nouveau, puis cliquez sur Document vierge.

3.     Création d'un nouveau document vierge

Pour créer un nouveau document à partir d'un modèle :

Un modèle est un document préconçu que vous pouvez utiliser pour créer rapidement un nouveau document. Les modèles incluent souvent une mise en forme et des designs personnalisés, ce qui peut vous faire gagner beaucoup de temps et d'efforts lorsque vous démarrez un nouveau projet.

 

1.    Cliquez sur l'onglet Fichier pour accéder à la vue Backstage, puis sélectionnez Nouveau.

2.    Plusieurs modèles apparaîtront sous l'option Document vierge. Vous pouvez également utiliser la barre de recherche pour trouver quelque chose de plus spécifique. Dans notre exemple, nous allons rechercher un modèle de flyer.

3.    Lorsque vous trouvez quelque chose qui vous plaît, sélectionnez un modèle pour le prévisualiser.

4.    Une prévisualisation du modèle apparaîtra. Cliquez sur Créer pour utiliser le modèle sélectionné.

5.    Un nouveau document apparaîtra avec le modèle sélectionné.

Après avoir effectué une recherche, vous pouvez également parcourir les modèles par catégorie.

Pour ouvrir un document existant :

En plus de créer de nouveaux documents, vous aurez souvent besoin d'ouvrir un document qui a été précédemment enregistré. Pour en savoir plus sur l'enregistrement de documents, consultez notre leçon sur l'enregistrement et le partage de documents.

 

1.    Accédez à la vue Backstage, puis cliquez sur Ouvrir, en cliquant sur la commande Ouvrir dans la vue Backstage

2.    Sélectionnez Ce PC, puis cliquez sur Parcourir. Vous pouvez également choisir OneDrive pour ouvrir des fichiers stockés sur votre OneDrive.

3.    La boîte de dialogue Ouvrir apparaîtra. Localisez et sélectionnez votre document, puis cliquez sur Ouvrir.

4.    Ouverture d'un fichier existant

Le document sélectionné apparaîtra.

La plupart des fonctionnalités de Microsoft Office, y compris Word, sont axées sur l'enregistrement et le partage de documents en ligne. Cela se fait avec OneDrive, qui est un espace de stockage en ligne pour vos documents et fichiers. Si vous souhaitez utiliser OneDrive, assurez-vous d'être connecté à Word avec votre compte Microsoft. Consultez notre leçon sur la compréhension de OneDrive pour en savoir plus.

Pour épingler un document :

Si vous travaillez fréquemment avec le même document, vous pouvez l'épingler dans la vue Backstage pour un accès rapide.

 

1.     Accédez à la vue Backstage, cliquez sur Ouvrir, puis sélectionnez Récent.

2.     Une liste de documents récemment modifiés s'affichera. Passez la souris sur le document que vous souhaitez épingler, puis cliquez sur l'icône de punaise.

3.     Le document restera dans la liste des documents récents jusqu'à ce qu'il soit désépinglé. Pour désépingler un document, cliquez à nouveau sur l'icône de punaise.

Mode de compatibilité

Parfois, vous pouvez avoir besoin de travailler avec des documents créés dans des versions antérieures de Microsoft Word, telles que Word 2010 ou Word 2007. Lorsque vous ouvrez ces types de documents, ils apparaissent en mode de compatibilité.

Le mode de compatibilité désactive certaines fonctionnalités, vous permettant d'accéder uniquement aux commandes disponibles dans le programme qui a été utilisé pour créer le document. Par exemple, si vous ouvrez un document créé dans Word 2007, vous ne pourrez utiliser que les onglets et les commandes disponibles dans Word 2007.

 

Dans l'image ci-dessous, vous pouvez voir comment le mode de compatibilité peut affecter les commandes disponibles. Parce que le document de gauche est en mode de compatibilité, il ne montre que les commandes qui étaient disponibles dans Word 2007.

 

Pour quitter le mode de compatibilité, vous devrez convertir le document en un type de version actuelle. Cependant, si vous collaborez avec d'autres personnes qui n'ont accès qu'à une version antérieure de Word, il est préférable de laisser le document en mode de compatibilité afin que le format ne change pas.

 

Vous pouvez consulter cette page de support de Microsoft pour en savoir plus sur les fonctionnalités désactivées en mode de compatibilité.

Pour convertir un document :

Si vous souhaitez accéder à toutes les fonctionnalités de Word 2016, vous pouvez convertir le document au format de fichier 2016.

 

1.    Cliquez sur l'onglet Fichier pour accéder à la vue Backstage, puis localisez et sélectionnez la commande Convertir.

2.    Une boîte de dialogue apparaîtra. Cliquez sur OK pour confirmer la mise à niveau du fichier.

 

3.    Le document sera converti au nouveau type de fichier.

 

La conversion d'un fichier peut entraîner quelques modifications de la mise en page originale du document.
