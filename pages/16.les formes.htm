Introduction

Vous pouvez ajouter une variété de formes à votre document, notamment des flèches, des bulles, des carrés, des étoiles et des formes de diagramme de flux. Vous souhaitez démarquer votre nom et votre adresse du reste de votre CV ? Utilisez une ligne. Vous avez besoin de créer un diagramme montrant une chronologie ou un processus ? Utilisez des formes de diagramme de flux. Bien que vous n'ayez pas besoin de formes dans chaque document que vous créez, elles peuvent ajouter de l'attrait visuel et de la clarté.

Pour insérer une forme :

1. Sélectionnez l'onglet Insérer, puis cliquez sur la commande Formes. Un menu déroulant de formes apparaîtra.

2. Sélectionnez la forme désirée.

   Sélection d'une forme

3. Cliquez et faites glisser à l'emplacement souhaité pour ajouter la forme à votre document.

   Insérer une nouvelle forme

4. Si vous le souhaitez, vous pouvez saisir du texte dans une forme. Lorsque la forme apparaît dans votre document, vous pouvez commencer à taper. Vous pouvez ensuite utiliser les options de mise en forme de l'onglet Accueil pour modifier la police, la taille de police ou la couleur du texte.

Pour changer l'ordre des formes :

Si une forme chevauche une autre, vous devrez peut-être changer l'ordre pour que la forme correcte apparaisse devant. Vous pouvez amener une forme vers l'avant ou l'envoyer vers l'arrière. Si vous avez plusieurs images, vous pouvez utiliser "Amener à l'avant" ou "Envoyer vers l'arrière" pour affiner l'ordre. Vous pouvez également déplacer une forme devant ou derrière du texte.

 

1. Faites un clic droit sur la forme que vous souhaitez déplacer. Dans notre exemple, nous voulons que le cœur apparaisse derrière le ruban, nous ferons donc un clic droit sur le cœur.

   Clic droit sur la forme

2. Dans le menu qui apparaît, passez le curseur sur "Amener à l'avant" ou "Envoyer vers l'arrière". Plusieurs options de classement apparaîtront. Sélectionnez l'option de classement souhaitée. Dans cet exemple, nous choisirons "Envoyer vers l'arrière".

   Sélection d'une option de classement

3. L'ordre des formes changera.

   Les formes réorganisées

4. Dans certains cas, l'option de classement que vous sélectionnez n'affectera pas l'ordre des formes. Si cela se produit, essayez de sélectionner à nouveau la même option ou essayez une option différente.

 

Si vous avez plusieurs formes superposées, il peut être difficile de sélectionner une forme individuellement. Le volet de sélection vous permet de sélectionner une forme et de la faire glisser vers un nouvel emplacement. Pour accéder au volet de sélection, cliquez sur "Volet de sélection" dans l'onglet Format.

Pour redimensionner une forme :

1. Sélectionnez la forme que vous souhaitez redimensionner. Des poignées de dimensionnement apparaîtront aux coins et sur les côtés de la forme.

   Cliquez sur la forme

2. Cliquez et faites glisser les poignées de dimensionnement jusqu'à ce que la forme atteigne la taille souhaitée. Vous pouvez utiliser les poignées de dimensionnement aux coins pour modifier à la fois la hauteur et la largeur de la forme en même temps.

   Redimensionnement de la forme

3. Pour faire pivoter la forme, cliquez et faites glisser la poignée de rotation.

   Rotation de la forme

4. Certaines formes ont également une ou plusieurs poignées jaunes qui peuvent être utilisées pour modifier la forme. Par exemple, avec les formes de bannière, vous pouvez ajuster la position des plis.

Modifier les formes

Word vous permet de modifier vos formes de diverses manières afin de les adapter à vos projets. Vous pouvez transformer une forme en une forme différente, formater le style et la couleur d'une forme, et ajouter divers effets.

Pour changer le style d'une forme :

Choisir un style de forme vous permet d'appliquer des couleurs et des effets prédéfinis pour changer rapidement l'apparence de votre forme.

 

1. Sélectionnez la forme que vous souhaitez changer.

   Sélection d'une forme

2. Dans l'onglet Format, cliquez sur la flèche déroulante "Plus" dans le groupe Styles de forme.

   Changer le style de forme

3. Un menu déroulant de styles apparaîtra. Sélectionnez le style que vous souhaitez utiliser.

   Sélection d'un style de forme

4. La forme apparaîtra dans le style sélectionné.

   La forme est formatée dans le nouveau style

Pour changer la couleur de remplissage d'une forme :

1. Sélectionnez la forme que vous souhaitez modifier.

   Sélection d'une forme

2. Dans l'onglet Format, cliquez sur la flèche déroulante Remplissage de forme. Sélectionnez la couleur que vous souhaitez utiliser. Pour afficher plus d'options de couleurs, sélectionnez Plus de couleurs de remplissage.

   Changer la couleur de remplissage de la forme

3. La forme apparaîtra dans la couleur de remplissage sélectionnée.

   La forme est formatée avec la nouvelle couleur de remplissage

4. Si vous souhaitez utiliser un type de remplissage différent, sélectionnez Dégradé ou Texture dans le menu déroulant. Vous pouvez également choisir Aucun remplissage pour le rendre transparent.

Pour changer le contour d'une forme :

1. Sélectionnez la forme que vous souhaitez modifier.

   Sélection d'une forme

2. Dans l'onglet Format, cliquez sur la flèche déroulante Contour de forme. Le menu Contour de forme apparaîtra.

   Changer la couleur du contour de la forme

3. Sélectionnez la couleur que vous souhaitez utiliser pour le contour. Si vous souhaitez rendre le contour transparent, sélectionnez "Aucun contour".

   La forme sera affichée dans la couleur de contour sélectionnée.

   La forme est formatée avec le nouveau contour

4. À partir du menu déroulant, vous pouvez modifier la couleur du contour, l'épaisseur (l'épaisseur) du contour et choisir s'il s'agit d'une ligne en pointillés.

Pour ajouter des effets à une forme :

1. Sélectionnez la forme que vous souhaitez modifier.

   Sélection d'une forme

2. Dans l'onglet Format, cliquez sur la flèche déroulante Effets de forme. Dans le menu qui apparaît, survolez le style d'effet que vous souhaitez ajouter, puis sélectionnez l'effet prédéfini souhaité.

   Sélection d'un style de biseau

3. La forme apparaîtra avec l'effet sélectionné.

   La forme avec l'effet sélectionné.

4. Pour ajuster davantage les effets de votre forme, sélectionnez "Options" à la fin de chaque menu. Le volet "Format de la forme" apparaîtra, vous permettant de personnaliser les effets.

Pour changer vers une forme différente :

1. Sélectionnez la forme que vous souhaitez changer. L'onglet Format apparaîtra.

   Sélection d'une forme

2. Dans l'onglet Format, cliquez sur la commande Modifier la forme. Dans le menu qui apparaît, passez la souris sur "Changer de forme", puis sélectionnez la forme souhaitée.

   Changer une forme

3. La nouvelle forme apparaîtra.

   La nouvelle forme
