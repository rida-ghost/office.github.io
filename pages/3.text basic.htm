Introduction

Si vous êtes nouveau dans Microsoft Word, vous devrez apprendre les bases de la saisie, de la modification et de l'organisation du texte. Les tâches de base comprennent la capacité d'ajouter, de supprimer et de déplacer du texte dans votre document, ainsi que la manière de couper, copier et coller.

Utilisation du point d'insertion pour ajouter du texte

Le point d'insertion est la ligne verticale clignotante dans votre document. Il indique l'endroit où vous pouvez entrer du texte sur la page. Vous pouvez utiliser le point d'insertion de différentes manières.

 

 Document vierge : Lorsqu'un nouveau document vierge s'ouvre, le point d'insertion apparaîtra dans le coin supérieur gauche de la page. Si vous le souhaitez, vous pouvez commencer à taper à partir de cet emplacement.

 Ajout d'espaces : Appuyez sur la barre d'espace pour ajouter des espaces après un mot ou entre du texte.

 Nouvelle ligne de paragraphe : Appuyez sur la touche Entrée de votre clavier pour déplacer le point d'insertion à la ligne de paragraphe suivante.

 Placement manuel : Une fois que vous commencez à taper, vous pouvez utiliser la souris pour déplacer le point d'insertion à un endroit spécifique de votre document. Cliquez simplement à l'endroit du texte où vous souhaitez le placer.

 Touches fléchées : Vous pouvez également utiliser les touches fléchées de votre clavier pour déplacer le point d'insertion. Les touches fléchées gauche et droite déplaceront entre les caractères adjacents sur la même ligne, tandis que les touches fléchées haut et bas déplaceront entre les lignes de paragraphe. Vous pouvez également appuyer sur Ctrl+Flèche gauche ou Ctrl+Flèche droite pour déplacer rapidement entre des mots entiers.

Dans un nouveau document vierge, vous pouvez double-cliquer sur la souris pour déplacer le point d'insertion ailleurs sur la page.

Sélectionner du texte

Avant de pouvoir déplacer ou mettre en forme du texte, vous devez d'abord le sélectionner. Pour ce faire, cliquez sur le texte avec votre souris et faites glisser, puis relâchez le bouton de la souris. Une boîte surlignée apparaîtra autour du texte sélectionné.

Lorsque vous sélectionnez du texte ou des images dans Word, une barre d'outils flottante avec des raccourcis de commandes apparaîtra. Si la barre d'outils ne s'affiche pas immédiatement, essayez de survoler la sélection avec la souris.

Pour sélectionner plusieurs lignes de texte :

 

1.    Déplacez le pointeur de la souris à gauche de n'importe quelle ligne pour qu'il devienne une flèche inclinée vers la droite.

2.    Placement du curseur à gauche du texte

3.    Cliquez avec la souris. La ligne sera sélectionnée.

4.    Pour sélectionner plusieurs lignes, cliquez et faites glisser la souris vers le haut ou vers le bas.

Sélection de plusieurs lignes de texte

 

Pour sélectionner l'ensemble du texte dans votre document, choisissez la commande Sélectionner dans l'onglet Accueil, puis cliquez sur Sélectionner tout. Vous pouvez également appuyer sur Ctrl+A sur votre clavier.

Sélection de tout le texte dans le document

D'autres raccourcis comprennent le double-clic pour sélectionner un mot et le triple-clic pour sélectionner une phrase ou un paragraphe entier.

Pour copier et coller du texte :

Sélectionnez le texte que vous souhaitez copier.

Cliquez sur la commande Copier dans l'onglet Accueil. Alternativement, vous pouvez appuyer sur Ctrl+C sur votre clavier.

Placez le point d'insertion à l'endroit où vous souhaitez que le texte apparaisse.

Sélectionnez l'emplacement où le texte copié apparaîtra

 

Cliquez sur la commande Coller dans l'onglet Accueil. Alternativement, vous pouvez appuyer sur Ctrl+V sur votre clavier.

Cliquez sur la commande Coller

 

Le texte sera alors collé à l'emplacement souhaité.

Pour couper et coller du texte :

 

1.      Sélectionnez le texte que vous souhaitez couper.

2.      Cliquez sur la commande Couper dans l'onglet Accueil. Alternativement, vous pouvez appuyer sur Ctrl+X sur votre clavier.

 

3.      Placez votre point d'insertion à l'endroit où vous souhaitez que le texte apparaisse.

4.      Cliquez sur la commande Coller dans l'onglet Accueil. Alternativement, vous pouvez appuyer sur Ctrl+V sur votre clavier.

5.      Le texte sera alors collé à l'emplacement souhaité.

Vous pouvez également couper, copier et coller en faisant un clic droit sur votre document et en choisissant l'action souhaitée dans le menu contextuel. Lorsque vous utilisez cette méthode pour coller, vous pouvez choisir parmi trois options qui déterminent comment le texte sera formaté : Conserver la mise en forme source, Fusionner la mise en forme et Conserver uniquement le texte. Vous pouvez survoler chaque icône pour voir à quoi elle ressemblera avant de la sélectionner.

Pour faire glisser et déposer du texte :

 

1.      Sélectionnez le texte que vous souhaitez déplacer.

2.      Cliquez sur le texte sélectionné, maintenez le bouton de la souris enfoncé et faites glisser le texte vers l'emplacement où vous souhaitez qu'il apparaisse. Un petit rectangle apparaîtra en dessous de la flèche pour indiquer que vous déplacez du texte.

 

3.      Relâchez le bouton de la souris, et le texte apparaîtra à l'emplacement choisi.

Si le texte n'apparaît pas exactement à l'emplacement souhaité, vous pouvez appuyer sur la touche Entrée de votre clavier pour déplacer le texte vers une nouvelle ligne.

Annuler et Rétablir

Disons que vous travaillez sur un document et que vous supprimez accidentellement du texte. Heureusement, vous n'aurez pas à retaper tout ce que vous venez de supprimer ! Word vous permet d'annuler votre action la plus récente lorsque vous faites une erreur comme celle-ci.

 

Pour ce faire, localisez et sélectionnez la commande Annuler sur la barre d'outils Accès rapide. Alternativement, vous pouvez appuyer sur Ctrl+Z sur votre clavier. Vous pouvez continuer à utiliser cette commande pour annuler plusieurs changements consécutifs.

En revanche, la commande Rétablir vous permet d'inverser la dernière annulation. Vous pouvez également accéder à cette commande en appuyant sur Ctrl+Y sur votre clavier.

Symboles

Si vous avez besoin d'insérer un caractère inhabituel qui ne se trouve pas sur votre clavier, comme un symbole de copyright (©) ou de marque déposée (™), vous pouvez généralement les trouver avec la commande Symbole.

 

Pour insérer un symbole :

 

1.      Placez le point d'insertion à l'endroit où vous souhaitez que le symbole apparaisse.

2.      Cliquez sur l'onglet Insertion.

 

3.      Localisez et sélectionnez la commande Symbole, puis choisissez le symbole souhaité dans le menu déroulant. Si vous ne trouvez pas celui que vous voulez, sélectionnez Plus de symboles.

4.      Le symbole apparaîtra dans le document.
