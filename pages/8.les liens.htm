Introduction

Ajouter des hyperliens au texte peut permettre d'accéder directement aux sites web et aux adresses e-mail depuis votre document. Il existe plusieurs façons d'insérer un hyperlien dans votre document. Selon l'apparence que vous souhaitez donner au lien, vous pouvez utiliser la mise en forme automatique des liens de Word ou convertir du texte en lien.

Comprendre les hyperliens dans Word

Les hyperliens sont composés de deux parties essentielles : l'adresse (URL) de la page web et le texte affiché. Par exemple, l'adresse peut être http://www.popsci.com, et le texte affiché peut être "Popular Science Magazine". Lorsque vous créez un hyperlien dans Word, vous avez la possibilité de choisir à la fois l'adresse et le texte affiché.

 

Word reconnaît souvent automatiquement les adresses e-mail et les adresses web lorsque vous les tapez, et les formate automatiquement en hyperliens après avoir appuyé sur Entrée ou sur la barre d'espace. Dans l'image ci-dessous, vous pouvez voir une adresse web hyperliée.

Pour suivre un hyperlien dans Word, maintenez la touche Ctrl enfoncée et cliquez sur l'hyperlien.

Pour formater un texte avec un hyperlien :

1. Sélectionnez le texte que vous souhaitez formater comme un hyperlien.

2. Sélectionnez l'onglet "Insertion", puis cliquez sur la commande "Hyperlien".

 Alternativement, vous pouvez ouvrir la boîte de dialogue "Insérer un hyperlien" en faisant un clic droit sur le texte sélectionné et en choisissant "Hyperlien..." dans le menu qui apparaît.

 

4. La boîte de dialogue "Insérer un hyperlien" s'affichera. À l'aide des options situées du côté gauche, vous pouvez choisir de créer un lien vers un fichier, une page web, une adresse e-mail, un document ou un emplacement dans le document actuel.

 

5. Le texte sélectionné apparaîtra dans le champ "Texte à afficher :" en haut. Vous pouvez modifier ce texte si vous le souhaitez.

 

6. Dans le champ "Adresse :", saisissez l'adresse à laquelle vous souhaitez créer un lien, puis cliquez sur OK.

7. Le texte sera alors formaté comme un hyperlien.

Après avoir créé un hyperlien, vous devriez le tester. Si vous avez créé un lien vers un site web, votre navigateur web devrait s'ouvrir automatiquement et afficher le site. Si cela ne fonctionne pas, vérifiez l'adresse de l'hyperlien pour d'éventuelles fautes de frappe.

Édition et suppression des hyperliens

Une fois que vous avez inséré un hyperlien, vous pouvez faire un clic droit sur l'hyperlien pour l'éditer, l'ouvrir, le copier ou le supprimer.

Pour supprimer un hyperlien, faites un clic droit sur l'hyperlien, puis sélectionnez "Supprimer l'hyperlien" dans le menu qui apparaît.
