Introduction

Un graphique est un outil que vous pouvez utiliser pour communiquer des informations de manière graphique. Inclure un graphique dans votre document peut vous aider à illustrer des données numériques, telles que des comparaisons et des tendances, afin qu'il soit plus facile pour le lecteur de comprendre.

 

Facultatif : Téléchargez notre document d'exercice.

Types de graphiques

Il existe plusieurs types de graphiques parmi lesquels choisir. Pour utiliser les graphiques efficacement, vous devrez comprendre ce qui les rend uniques.

 

Cliquez sur les flèches dans le diaporama ci-dessous pour en savoir plus sur les types de graphiques dans Word.

Diapositive 1

Word propose une variété de types de graphiques, chacun ayant ses propres avantages. Cliquez sur les flèches pour voir certains des différents types de graphiques disponibles dans Word.

Diapositive 1

Les graphiques à colonnes utilisent des barres verticales pour représenter les données. Ils peuvent fonctionner avec de nombreux types de données différents, mais ils sont le plus souvent utilisés pour comparer des informations.

Diapositive 1

Les graphiques linéaires sont idéaux pour montrer les tendances. Les points de données sont reliés par des lignes, ce qui facilite la visualisation de l'augmentation ou de la diminution des valeurs au fil du temps.

 

Diapositive 1

Les graphiques circulaires facilitent la comparaison des proportions. Chaque valeur est représentée comme une part du gâteau, il est donc facile de voir quelles valeurs constituent le pourcentage d'un tout.

Diapositive 1

Les graphiques à barres fonctionnent comme les graphiques à colonnes, mais ils utilisent des barres horizontales au lieu de verticales.

Diapositive 1

Les graphiques à aires sont similaires aux graphiques linéaires, à la différence que les zones sous les lignes sont remplies.

Diapositive 1

Les graphiques de surface vous permettent d'afficher des données sur un paysage en 3D. Ils fonctionnent mieux avec de grands ensembles de données, vous permettant de voir une variété d'informations en même temps.

Insérer des graphiques

Word utilise une fenêtre de feuille de calcul séparée pour entrer et éditer les données du graphique, un peu comme une feuille de calcul dans Excel. Le processus d'entrée de données est assez simple, mais si vous n'êtes pas familier avec Excel, vous voudrez peut-être consulter notre leçon sur les bases des cellules.

 

Pour insérer un graphique :

1. Placez le point d'insertion à l'endroit où vous souhaitez que le graphique apparaisse.

2. Accédez à l'onglet Insertion, puis cliquez sur la commande Graphique dans le groupe Illustrations.

   ![commande de graphique](https://image.url)

3. Une boîte de dialogue apparaîtra. Pour voir vos options, choisissez un type de graphique dans le volet de gauche, puis parcourez les graphiques sur la droite.

4. Sélectionnez le graphique souhaité, puis cliquez sur OK.

   ![sélection d'un style de graphique](https://image.url)

5. Une fenêtre de graphique et de feuille de calcul apparaîtra. Le texte dans la feuille de calcul n'est qu'un espace réservé que vous devrez remplacer par vos propres données sources. Les données sources sont ce que Word utilisera pour créer le graphique.

6. Saisissez vos données sources dans la feuille de calcul.

   ![saisie des données du graphique](https://image.url)

7. Seules les données incluses dans la zone bleue apparaîtront dans le graphique. Si nécessaire, cliquez et faites glisser le coin inférieur droit de la zone bleue pour augmenter ou réduire manuellement la plage de données.

   ![extension de la plage de données](https://image.url)

8. Lorsque vous avez terminé, cliquez sur X pour fermer la fenêtre de la feuille de calcul.

   ![fermeture de la fenêtre de la feuille de calcul](https://image.url)

9. Le graphique sera terminé.

   ![graphique terminé](https://image.url)

 

Pour modifier à nouveau votre graphique, il vous suffit de le sélectionner, puis de cliquer sur la commande Modifier les données dans l'onglet Création. La fenêtre de la feuille de calcul réapparaîtra.

Création de graphiques avec des données Excel existantes

Si vous avez déjà des données dans un fichier Excel existant que vous souhaitez utiliser dans Word, vous pouvez les copier et les coller au lieu de les saisir manuellement. Il vous suffit d'ouvrir la feuille de calcul dans Excel, de copier les données, puis de les coller en tant que données source dans Word.

 

Vous pouvez également incorporer un graphique Excel existant dans votre document Word. Cela est utile si vous savez que vous allez mettre à jour votre fichier Excel ultérieurement ; le graphique dans Word se mettra automatiquement à jour chaque fois qu'une modification sera apportée.

 

Consultez notre guide sur l'incorporation d'un graphique Excel pour plus d'informations.

Modifier les graphiques avec les outils de graphique

Il existe de nombreuses façons de personnaliser et d'organiser votre graphique dans Word. Par exemple, vous pouvez rapidement changer le type de graphique, réorganiser les données et même modifier l'apparence du graphique.

Pour inverser les données de lignes et de colonnes :

Parfois, vous voudrez peut-être modifier la manière dont vos données de graphique sont regroupées. Par exemple, dans le graphique ci-dessous, les données sont regroupées par genre, avec des colonnes pour chaque mois. Si nous inversions les lignes et les colonnes, les données seraient regroupées par mois à la place. Dans les deux cas, le graphique contient les mêmes données, mais elles sont présentées de manière différente.

 

1. Sélectionnez le graphique que vous souhaitez modifier. L'onglet Création apparaîtra sur le côté droit du ruban.

   ![sélection d'un graphique](https://image.url)

2. À partir de l'onglet Création, cliquez sur la commande Modifier les données dans le groupe Données.

   ![modification des données](https://image.url)

3. Cliquez à nouveau sur le graphique pour le sélectionner, puis cliquez sur la commande Inverser lignes/colonnes.

   ![inversion de lignes/colonnes](https://image.url)

4. Les lignes et les colonnes seront inversées. Dans notre exemple, les données sont désormais regroupées par mois, avec des colonnes pour chaque genre.

   ![graphique complet](https://image.url)

Pour changer le type de graphique :

Si vous constatez que le type de graphique que vous avez choisi ne convient pas à vos données, vous pouvez le changer pour un autre. Dans notre exemple, nous allons changer le type de graphique d'un graphique à colonnes à un graphique linéaire.

 

1. Sélectionnez le graphique que vous souhaitez modifier. L'onglet Création apparaîtra.

   ![sélection d'un graphique](https://image.url)

2. À partir de l'onglet Création, cliquez sur la commande Changer de type de graphique.

   ![modification du type de graphique](https://image.url)

3. Une boîte de dialogue apparaîtra. Sélectionnez le graphique souhaité, puis cliquez sur OK.

   ![sélection d'un graphique](https://image.url)

4. Le nouveau type de graphique sera appliqué. Dans notre exemple, le graphique linéaire facilite la visualisation des tendances au fil du temps.

   ![graphique linéaire](https://image.url)

Pour changer la disposition du graphique :

Pour modifier l'agencement de votre graphique, essayez de choisir une disposition différente. La disposition peut affecter plusieurs éléments, notamment le titre du graphique et les étiquettes de données.

 

1. Sélectionnez le graphique que vous souhaitez modifier. L'onglet Création apparaîtra.

   ![sélection d'un graphique](https://image.url)

2. À partir de l'onglet Création, cliquez sur la commande Disposition rapide.

   ![modification de la disposition](https://image.url)

3. Choisissez la disposition souhaitée dans le menu déroulant.

   ![sélection de la disposition](https://image.url)

4. Le graphique sera mis à jour pour refléter la nouvelle disposition.

   ![graphique complet](https://image.url)

 

Si vous ne trouvez pas une disposition de graphique qui correspond exactement à vos besoins, vous pouvez cliquer sur la commande Ajouter un élément de graphique de l'onglet Création pour ajouter des titres d'axe, des lignes de quadrillage et d'autres éléments de graphique.

 

Pour remplir un espace réservé (tel que le titre du graphique ou le titre de l'axe), cliquez sur l'élément et saisissez votre texte.

Pour changer le style du graphique :

Les styles de graphique de Word vous offrent un moyen simple de modifier la conception de votre graphique, y compris la couleur, le style et certains éléments de mise en page.

 

1. Sélectionnez le graphique que vous souhaitez modifier. L'onglet Création apparaîtra.

   ![sélection d'un graphique](https://image.url)

2. À partir de l'onglet Création, cliquez sur la flèche déroulante Plus dans le groupe Styles de graphique.

   ![menu Plus](https://image.url)

3. Un menu déroulant de styles apparaîtra. Sélectionnez le style que vous souhaitez.

   ![sélection d'un style de graphique](https://image.url)

4. Le style de graphique sera appliqué.

   ![graphique complet](https://image.url)

 

Pour une personnalisation encore plus rapide, utilisez les raccourcis de mise en forme situés à droite de votre graphique. Ils vous permettent d'ajuster le style du graphique, les éléments du graphique et même d'ajouter des filtres à vos données.
