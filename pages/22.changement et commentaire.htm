Introduction

Imaginons que quelqu'un vous demande de relire ou de collaborer sur un document. Si vous aviez une copie imprimée, vous pourriez utiliser un stylo rouge pour barrer des phrases, signaler des fautes d'orthographe et ajouter des commentaires dans les marges. Word vous permet de faire toutes ces choses électroniquement en utilisant les fonctionnalités Suivi des modifications et Commentaires.

Comprendre le Suivi des modifications

Lorsque vous activez le Suivi des modifications, chaque modification que vous apportez au document apparaîtra sous forme de mise en forme colorée. Si vous supprimez du texte, il ne disparaîtra pas, mais le texte sera barré. Si vous ajoutez du texte, il sera souligné. Cela vous permet de voir les modifications avant de les rendre définitives.

 

![modifications suivies dans un document](https://image.url)

 

Si plusieurs réviseurs interviennent, chaque personne se verra attribuer une couleur de mise en forme différente.

Pour activer le Suivi des modifications :

1. À partir de l'onglet Révision, cliquez sur la commande Suivi des modifications.

   ![activer le Suivi des modifications](https://image.url)

2. Le Suivi des modifications sera activé. À partir de ce moment, toutes les modifications que vous apportez au document apparaîtront sous forme de mises en forme colorées.

   ![modifications suivies dans un document](https://image.url)

Examiner les modifications

Les modifications suivies ne sont en réalité que des modifications suggérées. Pour devenir permanentes, elles doivent être acceptées. En revanche, l'auteur original peut ne pas être d'accord avec certaines des modifications suivies et choisir de les rejeter.

Pour accepter ou rejeter les modifications :

1. Sélectionnez la modification que vous souhaitez accepter ou rejeter.

   ![sélection d'une modification à accepter ou rejeter](https://image.url)

2. À partir de l'onglet Révision, cliquez sur la commande Accepter ou Rejeter.

   ![accepter une modification](https://image.url)

3. La mise en forme disparaîtra, et Word passera automatiquement à la modification suivante. Vous pouvez continuer à accepter ou rejeter chaque modification jusqu'à ce que vous les ayez toutes examinées.

   ![passer à la modification suivante](https://image.url)

4. Lorsque vous avez terminé, cliquez sur la commande Suivi des modifications pour désactiver le Suivi des modifications.

   ![désactiver le Suivi des modifications](https://image.url)

Pour accepter toutes les modifications en une seule fois, cliquez sur la flèche déroulante Accepter, puis sélectionnez Accepter tout. Si vous ne souhaitez plus suivre vos modifications, vous pouvez sélectionner Accepter tout et Arrêter de suivre.

Options d'affichage des modifications suivies

Si vous avez beaucoup de modifications suivies, elles peuvent devenir distractives lorsque vous essayez de lire un document. Heureusement, Word propose quelques moyens de personnaliser l'apparence des modifications suivies :

 

- Marque de révision simplifiée : Cela montre la version finale sans les marques en ligne. Les marqueurs rouges apparaîtront dans la marge gauche pour indiquer l'endroit où une modification a été apportée.

- Toutes les marques : Cela montre la version finale avec les marques en ligne.

- Aucune marque : Cela montre la version finale et masque toutes les marques.

- Originale : Cela montre la version originale et masque toutes les marques.

Pour masquer les modifications suivies :

1. À partir de l'onglet Révision, cliquez sur la commande Afficher pour la révision. La commande Afficher pour la révision se trouve à droite de la commande Suivi des modifications.

   ![cliquer sur la commande Afficher pour la révision](https://image.url)

2. Sélectionnez l'option souhaitée dans le menu déroulant. Dans notre exemple, nous sélectionnerons Aucune marque pour prévisualiser la version finale du document avant d'accepter les modifications.

   ![masquer les marques](https://image.url)

Vous pouvez également cliquer sur le marqueur dans la marge gauche pour basculer entre la Marque de révision simplifiée et Toutes les marques.

N'oubliez pas que masquer le Suivi des modifications n'équivaut pas à examiner les modifications. Vous devrez toujours accepter ou rejeter les modifications avant d'envoyer la version finale de votre document.

Pour afficher les révisions dans des bulles :

Par défaut, la plupart des révisions apparaissent en ligne, c'est-à-dire que le texte lui-même est marqué. Vous pouvez également choisir d'afficher les révisions dans des bulles, ce qui déplace la plupart des révisions vers la marge de droite. La suppression des marques en ligne peut rendre le document plus facile à lire, et les bulles vous donnent également des informations plus détaillées sur certaines marques.

 

1. À partir de l'onglet Révision, cliquez sur Afficher la mise en forme > Bulles > Afficher les révisions dans des bulles.

   ![activer les révisions dans des bulles](https://image.url)

2. La plupart des révisions apparaîtront dans la marge de droite, bien que tout texte ajouté apparaisse toujours en ligne.

Pour revenir aux marques en ligne, vous pouvez sélectionner soit Afficher toutes les révisions en ligne, soit Afficher uniquement les commentaires et la mise en forme dans des bulles.

Commentaires

Parfois, vous souhaitez ajouter un commentaire pour fournir des commentaires plutôt que de modifier un document. Bien qu'il soit souvent utilisé en combinaison avec le Suivi des modifications, vous n'avez pas nécessairement besoin d'activer le Suivi des modifications pour ajouter des commentaires.

Pour ajouter des commentaires :

1.       Sélectionnez du texte en le surlignant ou placez le curseur à l'endroit où vous souhaitez que le commentaire apparaisse.

2.      

3.       À partir de l'onglet Révision, cliquez sur la commande Nouveau commentaire.

3. Saisissez votre commentaire. Lorsque vous avez terminé, vous pouvez fermer la boîte de commentaire en appuyant sur la touche Échap ou en cliquant n'importe où à l'extérieur de la boîte de commentaire.

   ![commentaire ajouté](https://image.url)

Pour supprimer des commentaires :

1. Sélectionnez le commentaire que vous souhaitez supprimer.

   ![sélection d'un commentaire](https://image.url)

2. À partir de l'onglet Révision, cliquez sur la commande Supprimer.

   ![supprimer un commentaire](https://image.url)

 

3. Le commentaire sera supprimé.

  

Pour supprimer tous les commentaires, cliquez sur la flèche déroulante Supprimer et sélectionnez Supprimer tous les commentaires dans le document.

Comparer des documents

Si vous modifiez un document sans suivre les modifications, il est toujours possible d'utiliser des fonctionnalités de révision telles qu'Accepter et Rejeter. Vous pouvez le faire en comparant deux versions du document. Vous avez seulement besoin du document original et du document révisé (les documents doivent également avoir des noms de fichier différents).

Pour comparer deux documents :

1. À partir de l'onglet Révision, cliquez sur la commande Comparer, puis sélectionnez Comparer dans le menu déroulant.

   ![cliquer sur la commande Comparer dans l'onglet Révision](https://image.url)

2. Une boîte de dialogue apparaîtra. Choisissez votre document Original en cliquant sur la flèche déroulante et en sélectionnant le document dans la liste. Si le fichier n'est pas dans la liste, cliquez sur le bouton Parcourir pour le localiser.

   ![choisir les fichiers à comparer](https://image.url)

 

4.       Choisissez le document Révisé, puis cliquez sur OK.

5.       Word comparera les deux fichiers pour déterminer ce qui a été modifié, puis créera un nouveau document. Les modifications apparaîtront sous forme de mises en forme colorées, tout comme avec le Suivi des modifications. Vous pouvez ensuite utiliser les commandes Accepter et Rejeter pour finaliser le document.
