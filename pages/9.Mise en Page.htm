Introduction

Word offre une variété d'options de mise en page et de formatage qui affectent la manière dont le contenu apparaît sur la page. Vous pouvez personnaliser l'orientation de la page, la taille du papier et les marges de la page en fonction de la manière dont vous souhaitez que votre document apparaisse.

Orientation de la page

Word propose deux options d'orientation de page : paysage et portrait. Comparez notre exemple ci-dessous pour voir comment l'orientation peut influencer l'apparence et l'espacement du texte et des images.

 

·         Le paysage signifie que la page est orientée horizontalement.

·         Le portrait signifie que la page est orientée verticalement.

Pour changer l'orientation de la page :

1. Sélectionnez l'onglet Mise en page.

2. Cliquez sur la commande Orientation dans le groupe Configuration de la page.

3. Un menu déroulant apparaîtra. Cliquez sur Portrait ou Paysage pour modifier l'orientation de la page.

   4. L'orientation de la page du document sera modifiée.

Taille de la page

Par défaut, la taille de page d'un nouveau document est de 8,5 pouces sur 11 pouces. En fonction de votre projet, vous devrez peut-être ajuster la taille de la page de votre document. Il est important de noter qu'avant de modifier la taille de page par défaut, vous devriez vérifier quelles sont les tailles de page que votre imprimante peut prendre en charge.

 

Pour changer la taille de la page :

Word propose une variété de tailles de page prédéfinies parmi lesquelles choisir.

 

1.       Sélectionnez l'onglet Mise en page, puis cliquez sur la commande Taille.

2.       Un menu déroulant apparaîtra. La taille de page actuelle est mise en évidence. Cliquez sur la taille de page prédéfinie souhaitée.

3. La taille de page du document sera modifiée.

Pour utiliser une taille de page personnalisée :

Word vous permet également de personnaliser la taille de la page dans la boîte de dialogue Configuration de la page.

 

1. À partir de l'onglet Mise en page, cliquez sur Taille. Sélectionnez Plus de tailles de papier dans le menu déroulant.

2. La boîte de dialogue Configuration de la page s'affichera.

3.       Ajustez les valeurs de Largeur et Hauteur, puis cliquez sur OK.

4. La taille de page du document sera modifiée.

Marges de la page

Une marge est l'espace entre le texte et le bord de votre document. Par défaut, les marges d'un nouveau document sont définies sur Normal, ce qui signifie qu'il y a un espace d'un pouce entre le texte et chaque bord. En fonction de vos besoins, Word vous permet de modifier la taille des marges de votre document.

 

Pour formater les marges de la page :

Word propose une variété de tailles de marges prédéfinies parmi lesquelles choisir.

 

1. Sélectionnez l'onglet Mise en page, puis cliquez sur la commande Marges.

  

2. Un menu déroulant apparaîtra. Cliquez sur la taille de marge prédéfinie que vous souhaitez.

3. Les marges du document seront modifiées.

Pour utiliser des marges personnalisées :

Word vous permet également de personnaliser la taille de vos marges dans la boîte de dialogue Configuration de la page.

 

1. À partir de l'onglet Mise en page, cliquez sur Marges. Sélectionnez Marges personnalisées dans le menu déroulant.

2. La boîte de dialogue Configuration de la page s'affichera.

3. Ajustez les valeurs pour chaque marge, puis cliquez sur OK.

4. Les marges du document seront modifiées.

 

Alternativement, vous pouvez ouvrir la boîte de dialogue Configuration de la page en accédant à l'onglet Mise en page et en cliquant sur la petite flèche dans le coin inférieur droit du groupe Configuration de la page.

Ouverture de la boîte de dialogue Configuration de la page

Vous pouvez utiliser la fonction Prédéfinir comme par défaut de Word pour enregistrer toutes les modifications de format que vous avez apportées et les appliquer automatiquement aux nouveaux documents. Pour apprendre comment faire cela, lisez notre article sur la modification de vos paramètres par défaut dans Word.
