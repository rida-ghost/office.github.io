Introduction

Lorsque vous concevez votre document et prenez des décisions de mise en forme, vous devrez tenir compte de l'espacement entre les lignes et les paragraphes. Vous pouvez augmenter l'espacement pour améliorer la lisibilité et le réduire pour faire tenir plus de texte sur la page.

Au sujet de l'espacement entre les lignes

L'espacement entre les lignes correspond à l'espace entre chaque ligne d'un paragraphe. Word vous permet de personnaliser l'espacement entre les lignes pour qu'il soit simple (une ligne de hauteur), double (deux lignes de hauteur), ou toute autre valeur de votre choix. L'espacement par défaut dans Word est de 1,08 lignes, légèrement supérieur à un espacement simple.

 

Dans les images ci-dessous, vous pouvez comparer différents types d'espacement entre les lignes. De gauche à droite, ces images montrent l'espacement entre les lignes par défaut, l'espacement simple et l'espacement double.

Pour formater l'espacement entre les lignes :

 

1.       Sélectionnez le texte que vous souhaitez formater.

2.       Sur l'onglet Accueil, cliquez sur la commande Espacement entre les lignes et les paragraphes, puis sélectionnez l'espacement entre les lignes souhaité.

3.       L'espacement entre les lignes changera dans le document.

 

Pour affiner le réglage de l'espacement entre les lignes :

 

Vos options d'espacement entre les lignes ne se limitent pas à celles disponibles dans le menu Espacement entre les lignes et les paragraphes. Pour ajuster l'espacement avec plus de précision, sélectionnez Options d'espacement entre les lignes dans le menu pour accéder à la boîte de dialogue Paragraphe. Vous disposerez alors de quelques options supplémentaires pour personnaliser l'espacement.

 

Exactement : Lorsque vous choisissez cette option, l'espacement entre les lignes est mesuré en points, tout comme la taille de la police. Par exemple, si vous utilisez un texte de 12 points, vous pouvez utiliser un espacement de 15 points.

 

Au moins : Comme l'option "Exactement", celle-ci vous permet de choisir combien de points d'espacement vous souhaitez. Cependant, si vous avez des tailles de texte différentes sur la même ligne, l'espacement s'ajustera pour s'adapter au texte le plus grand.

 

Multiple : Cette option vous permet de saisir le nombre de lignes d'espacement souhaité. Par exemple, en choisissant "Multiple" et en réglant l'espacement à 1,2, le texte sera légèrement plus espacé que du texte à espacement simple. Si vous souhaitez que les lignes soient plus rapprochées, vous pouvez choisir une valeur plus petite, comme 0,9.

 

Espacement entre les paragraphes

Tout comme vous pouvez formater l'espacement entre les lignes de votre document, vous pouvez ajuster l'espacement avant et après les paragraphes. Cela est utile pour séparer les paragraphes, les titres et les sous-titres.

Pour formater l'espacement entre les paragraphes :

 

Dans notre exemple, nous allons augmenter l'espace avant chaque paragraphe pour les séparer un peu plus. Cela rendra la lecture un peu plus aisée.

 

1.       Sélectionnez le paragraphe ou les paragraphes que vous souhaitez formater.

2.       Sur l'onglet Accueil, cliquez sur la commande Espacement entre les lignes et les paragraphes. Cliquez sur "Ajouter de l'espace avant le paragraphe" ou "Supprimer l'espace après le paragraphe" dans le menu déroulant. Dans notre exemple, nous allons sélectionner "Ajouter de l'espace avant le paragraphe".

3.       L'espacement entre les paragraphes changera dans le document.

À partir du menu déroulant, vous pouvez également sélectionner "Options d'espacement entre les lignes" pour ouvrir la boîte de dialogue Paragraphe. À partir de là, vous pouvez contrôler la quantité d'espace avant et après le paragraphe.

Vous pouvez utiliser la fonctionnalité pratique "Définir par défaut" de Word pour enregistrer toutes les modifications de mise en forme que vous avez apportées et les appliquer automatiquement aux nouveaux documents. Pour apprendre comment faire cela, consultez notre article sur la modification de vos paramètres par défaut dans Word.
