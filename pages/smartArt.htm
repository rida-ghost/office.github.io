Introduction

SmartArt vous permet de communiquer des informations à l'aide de graphiques plutôt que de simplement utiliser du texte. Il existe une variété de styles parmi lesquels vous pouvez choisir pour illustrer de nombreuses idées différentes.

Pour insérer un graphique SmartArt :

1. Placez le point d'insertion dans le document à l'endroit où vous souhaitez que le graphique SmartArt apparaisse.

2. Dans l'onglet Insertion, sélectionnez la commande SmartArt dans le groupe Illustrations.

3. En sélectionnant la commande SmartArt dans l'onglet Insertion.

4. Une boîte de dialogue apparaîtra. Sélectionnez une catégorie à gauche, choisissez le graphique SmartArt désiré, puis cliquez sur OK.

5. En choisissant un graphique SmartArt.

6. Le graphique SmartArt apparaîtra dans votre document.

7. Le SmartArt inséré.

Pour ajouter du texte à un graphique SmartArt :

1.       Sélectionnez le graphique SmartArt. Le volet de texte devrait apparaître sur le côté gauche. S'il ne s'affiche pas, vous pouvez cliquer sur la petite flèche sur le bord gauche du graphique.

2. Saisissez du texte à côté de chaque puce dans le volet de texte. Le texte apparaîtra dans la forme correspondante et sera automatiquement redimensionné pour s'adapter à l'intérieur de la forme.

3. Ajouter du texte au graphique SmartArt.

4. Vous pouvez également ajouter du texte en cliquant sur la forme désirée, puis en tapant. Cela fonctionne bien si vous devez ajouter du texte à seulement quelques formes. Cependant, pour des graphiques SmartArt plus complexes, travailler dans le volet de texte est souvent plus rapide et plus facile.

Pour réorganiser, ajouter et supprimer des formes :

Il est facile d'ajouter de nouvelles formes, de changer leur ordre et même de supprimer des formes de votre graphique SmartArt. Vous pouvez effectuer toutes ces actions dans le volet de texte, et cela ressemble beaucoup à la création d'une structure avec une liste multiniveau. Pour plus d'informations sur les listes multilignes, vous voudrez peut-être consulter notre leçon sur les listes.

 

Pour rétrograder une forme, sélectionnez la puce désirée, puis appuyez sur la touche Tab. La puce se déplacera vers la droite, et la forme descendra d'un niveau.

Rétrograder une forme dans le volet de texte.

Pour promouvoir une forme, sélectionnez la puce désirée, puis appuyez sur la touche Retour arrière (ou Shift+Tab). La puce se déplacera vers la gauche, et la forme montera d'un niveau.

Promouvoir une forme.

Pour ajouter une nouvelle forme, placez le point d'insertion après la puce désirée, puis appuyez sur Entrée. Une nouvelle puce apparaîtra dans le volet de texte, et une nouvelle forme apparaîtra dans le graphique.

Ajouter une forme au graphique.

Pour supprimer une forme, continuez d'appuyer sur la touche Retour arrière jusqu'à ce que la puce soit supprimée. La forme sera alors supprimée. Dans notre exemple, nous allons supprimer toutes les formes sans texte.

Pour organiser SmartArt depuis l'onglet Création

Si vous préférez ne pas utiliser le volet de texte pour organiser votre SmartArt, vous pouvez utiliser les commandes de l'onglet Création dans le groupe Créer un graphique. Sélectionnez simplement la forme que vous souhaitez modifier, puis choisissez la commande désirée.

 

Promouvoir et Rétrograder : Utilisez ces commandes pour déplacer une forme vers le haut ou vers le bas entre les niveaux.

Les commandes Promouvoir et Rétrograder.

Monter et Descendre : Utilisez ces commandes pour changer l'ordre des formes sur le même niveau.

Les commandes Monter et Descendre.

Ajouter une forme : Utilisez cette commande pour ajouter une nouvelle forme à votre graphique. Vous pouvez également cliquer sur la flèche déroulante pour plus d'options de placement précis.

Cliquez sur la commande Ajouter une forme pour ouvrir le menu déroulant.

Dans notre exemple, nous avons organisé un graphique avec une disposition hiérarchique. Tous les graphiques SmartArt n'utilisent pas ce type de disposition, alors gardez à l'esprit que ces commandes peuvent fonctionner différemment (ou pas du tout) en fonction de la disposition de votre graphique.

Personnalisation de SmartArt

Après avoir inséré SmartArt, il y a plusieurs choses que vous pourriez vouloir changer concernant son apparence. Chaque fois que vous sélectionnez un graphique SmartArt, les onglets Conception et Format apparaissent sur le côté droit du Ruban. À partir de là, il est facile de modifier le style et la mise en page d'un graphique SmartArt.

 

Il existe plusieurs styles SmartArt qui vous permettent de modifier rapidement l'apparence de votre SmartArt. Pour changer le style, sélectionnez le style désiré dans le groupe Styles SmartArt.

Choisir un style SmartArt.

Vous avez une variété de schémas de couleurs à utiliser avec SmartArt. Pour changer les couleurs, cliquez sur la commande Modifier les couleurs et choisissez l'option désirée dans le menu déroulant.

Changer les couleurs de SmartArt.

Pour changer la mise en page de SmartArt :

Si vous n'aimez pas la manière dont vos informations sont organisées dans un graphique SmartArt, vous pouvez toujours changer sa mise en page pour mieux adapter votre contenu.

 

1. Depuis l'onglet Conception, cliquez sur la flèche déroulante Plus dans le groupe Mises en page.

   ![Cliquez sur la flèche déroulante Plus](image-link)

2. Choisissez la mise en page désirée, ou cliquez sur Plus de Mises en page pour voir encore plus d'options.

   ![Choisissez une mise en page SmartArt différente](image-link)

3. La mise en page sélectionnée s'affichera.

   ![La nouvelle mise en page SmartArt](image-link)

Si la nouvelle mise en page est trop différente de l'originale, il se peut que certaines parties de votre texte ne s'affichent pas. Avant de choisir une nouvelle mise en page, vérifiez attentivement pour vous assurer qu'aucune information importante ne sera perdue.

 

Vous pouvez également personnaliser chaque forme indépendamment. Sélectionnez simplement n'importe quelle forme dans le graphique, puis choisissez l'option désirée dans l'onglet Format.
