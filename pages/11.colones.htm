Introduction

Parfois, les informations que vous incluez dans votre document sont mieux présentées en colonnes. Non seulement les colonnes peuvent améliorer la lisibilité, mais certains types de documents, tels que les articles de journaux, les newsletters et les dépliants, sont souvent rédigés au format de colonnes. Word vous permet également d'ajuster vos colonnes en ajoutant des sauts de colonne.

 

Facultatif : Téléchargez notre document d'entraînement.

Pour ajouter des colonnes à un document :

1.       Sélectionnez le texte que vous souhaitez formater.

2. Sélectionnez l'onglet Mise en page, puis cliquez sur la commande Colonnes. Un menu déroulant apparaîtra.

3. Choisissez le nombre de colonnes que vous souhaitez créer.

4. Le texte sera formaté en colonnes.

Vos choix de colonnes ne se limitent pas au menu déroulant qui apparaît. Sélectionnez Plus de colonnes en bas du menu pour accéder à la boîte de dialogue Colonnes. Cliquez sur les flèches à côté de Nombre de colonnes : pour ajuster le nombre de colonnes.

Si vous souhaitez ajuster l'espacement et l'alignement des colonnes, cliquez et faites glisser les repères d'indentation sur la règle jusqu'à ce que les colonnes apparaissent comme vous le souhaitez.

Pour supprimer des colonnes :

Pour supprimer la mise en forme en colonnes, placez le curseur d'insertion n'importe où dans les colonnes, puis cliquez sur la commande Colonnes dans l'onglet Mise en page. Sélectionnez "Une" dans le menu déroulant qui apparaît.

Ajout de sauts de colonne

Une fois que vous avez créé des colonnes, le texte s'écoulera automatiquement d'une colonne à l'autre. Cependant, parfois, vous souhaiterez peut-être contrôler précisément où chaque colonne commence. Vous pouvez le faire en créant un saut de colonne.

Pour ajouter un saut de colonne :

Dans notre exemple ci-dessous, nous allons ajouter un saut de colonne qui déplacera le texte au début de la colonne suivante.

 

1. Placez le curseur d'insertion au début du texte que vous souhaitez déplacer.

2. Sélectionnez l'onglet Mise en page, puis cliquez sur la commande Sauts. Un menu déroulant apparaîtra.

3. Sélectionnez "Colonnes" dans le menu.

4. Le texte se déplacera au début de la colonne. Dans notre exemple, il a été déplacé au début de la colonne suivante.

5. Pour en savoir plus sur l'ajout de sauts dans votre document, consultez notre leçon sur les Sauts.

 

Pour supprimer des sauts de colonne :

1.       Par défaut, les sauts sont masqués. Si vous souhaitez afficher les sauts dans votre document, cliquez sur la commande Afficher/Masquer dans l'onglet Accueil.

2.       Placez le curseur d'insertion à gauche du saut que vous souhaitez supprimer.

3.       Appuyez sur la touche Supprimer pour supprimer le saut.
