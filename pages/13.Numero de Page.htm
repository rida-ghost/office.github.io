Introduction

Les numéros de page peuvent être utilisés pour numéroter automatiquement chaque page de votre document. Ils sont disponibles dans une large gamme de formats de numérotation et peuvent être personnalisés selon vos besoins. Les numéros de page sont généralement placés dans l'en-tête, le pied de page ou la marge latérale. Lorsque vous avez besoin de numéroter certaines pages différemment, Word vous permet de recommencer la numérotation des pages.

Pour ajouter des numéros de page :

Word peut automatiquement étiqueter chaque page avec un numéro de page et le placer dans un en-tête, un pied de page ou une marge latérale. Si vous avez déjà un en-tête ou un pied de page existant, il sera supprimé et remplacé par le numéro de page.

 

1.       Sur l'onglet "Insertion", cliquez sur la commande "Numéro de page".

2.       La commande "Numéro de page" se trouve sur l'onglet "Insertion".

3.       Ouvrez le menu "Haut de page", "Bas de page" ou "Marge de page", selon l'endroit où vous souhaitez positionner le numéro de page. Ensuite, sélectionnez le style d'en-tête souhaité.

4.       Sélectionnez un style d'en-tête.

5.       Les numéros de page apparaîtront.

6. Appuyez sur la touche "Échap" pour verrouiller l'en-tête et le pied de page.

8. Si vous avez besoin d'apporter des modifications à vos numéros de page, double-cliquez simplement sur l'en-tête ou le pied de page pour le déverrouiller.

9. Si vous avez créé un numéro de page dans la marge latérale, il est toujours considéré comme faisant partie de l'en-tête ou du pied de page. Vous ne pourrez pas sélectionner le numéro de page à moins que l'en-tête ou le pied de page ne soit sélectionné.

Pour ajouter des numéros de page à un en-tête ou un pied de page existant :

Si vous avez déjà un en-tête ou un pied de page et que vous souhaitez y ajouter un numéro de page, Word propose une option pour insérer automatiquement le numéro de page dans l'en-tête ou le pied de page existant. Dans notre exemple, nous ajouterons une numérotation de page à l'en-tête de notre document.

 

1. Double-cliquez n'importe où sur l'en-tête ou le pied de page pour le déverrouiller.

   Déverrouillage de l'en-tête

2. Sur l'onglet "Création", cliquez sur la commande "Numéro de page". Dans le menu qui apparaît, survolez "Position actuelle" avec la souris et sélectionnez le style de numérotation de page souhaité.

   Le menu Numéro de page

3. La numérotation de page apparaîtra.

   Numérotation de page dans l'en-tête

4. Lorsque vous avez terminé, appuyez sur la touche Échap.

Pour masquer le numéro de page sur la première page :

Dans certains documents, vous ne souhaitez peut-être pas afficher le numéro de page sur la première page. Vous pouvez masquer le numéro de page sur la première page sans affecter les autres pages.

 

1. Double-cliquez sur l'en-tête ou le pied de page pour le déverrouiller.

2. Sur l'onglet "Création", cochez la case "Première page différente". L'en-tête et le pied de page disparaîtront de la première page. Si vous le souhaitez, vous pouvez saisir quelque chose de nouveau dans l'en-tête ou le pied de page, et cela n'affectera que la première page.

3. Cliquez sur "Première page différente".

 

Si vous ne parvenez pas à sélectionner "Première page différente", cela peut être dû au fait qu'un objet dans l'en-tête ou le pied de page est sélectionné. Cliquez dans une zone vide de l'en-tête ou du pied de page pour vous assurer que rien n'est sélectionné.

Pour recommencer la numérotation des pages :

Word vous permet de recommencer la numérotation des pages à n'importe quelle page de votre document. Vous pouvez le faire en insérant un saut de section et en sélectionnant le numéro avec lequel vous souhaitez recommencer la numérotation. Dans notre exemple, nous allons recommencer la numérotation des pages pour la section "Œuvres citées" de notre document.

 

1.       Placez le curseur d'insertion en haut de la page pour laquelle vous souhaitez recommencer la numérotation des pages. Si du texte est présent sur la page, placez le curseur d'insertion au début du texte.

2.       Sélectionnez l'onglet "Mise en page", puis cliquez sur la commande "Sauts". Sélectionnez "Saut de section suivante" dans le menu déroulant qui apparaît.

3. Un saut de section sera ajouté au document.

4. Double-cliquez sur l'en-tête ou le pied de page contenant le numéro de page que vous souhaitez recommencer.

6.       Cliquez sur la commande "Numéro de page". Dans le menu qui apparaît, sélectionnez "Format des numéros de page".

7.       Une boîte de dialogue apparaîtra. Cliquez sur le bouton "Commencer à partir de :" Par défaut, il commencera à 1. Si vous le souhaitez, vous pouvez changer le numéro. Lorsque vous avez terminé, cliquez sur OK.

8.      

7. La numérotation des pages recommencera.

8. Pour en savoir plus sur l'ajout de sauts de section à votre document, consultez notre leçon sur les Sauts.
