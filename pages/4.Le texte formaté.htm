Introduction :

Le texte formaté peut attirer l'attention du lecteur sur des parties spécifiques d'un document et mettre en évidence les informations importantes. Dans Word, vous avez plusieurs options pour ajuster le texte, notamment la police, la taille et la couleur. Vous pouvez également ajuster l'alignement du texte pour modifier sa présentation sur la page.

Pour changer la taille de la police :

 

1.     Sélectionnez le texte que vous souhaitez modifier.

2.     Sur l'onglet Accueil, cliquez sur la flèche déroulante Taille de police. Sélectionnez une taille de police dans le menu. Si la taille de police dont vous avez besoin n'est pas disponible dans le menu, vous pouvez cliquer dans la boîte Taille de police et taper la taille de police souhaitée, puis appuyer sur Entrée.

 

3.     La taille de la police changera dans le document.

Vous pouvez également utiliser les commandes Augmenter la taille de la police et Réduire la taille de la police pour modifier la taille de la police.

Pour changer la police :

Par défaut, la police de chaque nouveau document est définie sur Calibri. Cependant, Word propose de nombreuses autres polices que vous pouvez utiliser pour personnaliser le texte.

 

1.     Sélectionnez le texte que vous souhaitez modifier.

2.     Sur l'onglet Accueil, cliquez sur la flèche déroulante à côté de la boîte Police. Un menu de styles de police apparaîtra.

3.     Sélectionnez le style de police que vous souhaitez utiliser.

4.     La police changera dans le document.

Lors de la création d'un document professionnel ou d'un document comportant plusieurs paragraphes, il est préférable de choisir une police facile à lire. Outre Calibri, les polices couramment utilisées pour la lecture incluent Cambria, Times New Roman et Arial.

Pour changer la couleur de la police :

1.      Sélectionnez le texte que vous souhaitez modifier.

2.      Sur l'onglet Accueil, cliquez sur la flèche déroulante Couleur de police. Le menu Couleur de police apparaît.

3.      Sélectionnez la couleur de police que vous souhaitez utiliser. La couleur de la police changera dans le document.

Vos choix de couleurs ne se limitent pas au menu déroulant qui apparaît. Sélectionnez Plus de couleurs en bas du menu pour accéder à la boîte de dialogue Couleurs. Choisissez la couleur que vous souhaitez, puis cliquez sur OK.

Pour utiliser les commandes Gras, Italique et Souligné :

Les commandes Gras, Italique et Souligné peuvent être utilisées pour mettre en évidence des mots ou des phrases importantes.

 

1.     Sélectionnez le texte que vous souhaitez modifier.

2.     Sur l'onglet Accueil, cliquez sur les commandes Gras (B), Italique (I) ou Souligné (U) dans le groupe Police. Dans notre exemple, nous cliquerons sur Gras.

3.     Le texte sélectionné sera modifié dans le document.

 

Pour changer la casse du texte :

Lorsque vous avez besoin de modifier rapidement la casse d'un texte, vous pouvez utiliser la commande Changer la casse au lieu de supprimer et retaper le texte.

 

1.      Sélectionnez le texte que vous souhaitez modifier.

2.      Sur l'onglet Accueil, cliquez sur la commande Changer la casse dans le groupe Police.

3.      Un menu déroulant apparaîtra. Sélectionnez l'option de casse souhaitée dans le menu.

4.      La casse du texte sera modifiée dans le document.

Pour surligner du texte :

Le surlignage peut être un outil utile pour mettre en évidence du texte important dans votre document.

 

1.     Sélectionnez le texte que vous souhaitez surligner.

2.     Depuis l'onglet Accueil, cliquez sur la flèche déroulante Couleur de surlignage de texte. Le menu Couleur de surlignage apparaît.

 

3.     Sélectionnez la couleur de surlignage souhaitée. Le texte sélectionné sera alors surligné dans le document.

Pour supprimer le surlignage, sélectionnez le texte surligné, puis cliquez sur la flèche déroulante Couleur de surlignage de texte. Sélectionnez Aucune couleur dans le menu déroulant.

Si vous devez surligner plusieurs lignes de texte, changer le curseur en un surligneur peut être une alternative pratique à la sélection et au surlignage des lignes individuellement. Cliquez sur la commande Couleur de surlignage de texte, et le curseur se transforme en surligneur. Vous pouvez ensuite cliquer et faire glisser le surligneur sur les lignes que vous souhaitez surligner.

Pour changer l'alignement du texte :

Par défaut, Word aligne le texte sur la marge gauche dans les nouveaux documents. Cependant, il peut y avoir des moments où vous souhaitez ajuster l'alignement du texte au centre ou à droite.

 

1.      Sélectionnez le texte que vous souhaitez modifier.

2.      Sur l'onglet Accueil, sélectionnez l'une des quatre options d'alignement du groupe Paragraphe. Dans notre exemple, nous avons sélectionné l'Alignement au Centre.

3.      Le texte sera réaligné dans le document.

Cliquez sur les flèches dans le diaporama ci-dessous pour en savoir plus sur les quatre options d'alignement du texte.

Aligner le texte à gauche : Cela aligne tout le texte sélectionné sur la marge gauche. La commande Aligner le texte à gauche est l'alignement le plus courant et est sélectionnée par défaut lors de la création d'un nouveau document.

Centre : Cela aligne le texte à égale distance des marges gauche et droite.

Aligner le texte à droite : Cela aligne tout le texte sélectionné sur la marge droite.

Justifier : Le texte justifié est égal des deux côtés. Il s'aligne également sur les marges droite et gauche. De nombreux journaux et magazines utilisent la justification complète.

 

Vous pouvez utiliser la fonction Prédéfinir de Word pour enregistrer toutes les modifications de mise en forme que vous avez apportées et les appliquer automatiquement aux nouveaux documents. Pour apprendre comment faire cela, consultez notre article sur la modification de vos paramètres par défaut dans Word.
