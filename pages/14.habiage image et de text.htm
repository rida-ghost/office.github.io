Introduction

Ajouter des images à votre document peut être un excellent moyen d'illustrer des informations importantes et d'ajouter des accents décoratifs au texte existant. Utilisées avec modération, les images peuvent améliorer l'apparence générale de votre document.

 

Pour insérer une image à partir d'un fichier :

Si vous avez une image spécifique en tête, vous pouvez insérer une image à partir d'un fichier. Dans notre exemple, nous allons insérer une image enregistrée localement sur notre ordinateur. Si vous souhaitez suivre notre exemple, faites un clic droit sur l'image ci-dessous et enregistrez-la sur votre ordinateur.

Placez le point d'insertion où vous souhaitez que l'image apparaisse.

Sélectionnez l'onglet Insérer dans le Ruban, puis cliquez sur la commande Images.

La boîte de dialogue Insérer une image apparaîtra. Naviguez jusqu'au dossier où se trouve votre image, puis sélectionnez l'image et cliquez sur Insérer.

L'image apparaîtra dans le document.

Pour redimensionner une image, cliquez et faites glisser l'une des poignées de redimensionnement situées dans les coins. L'image changera de taille tout en conservant les mêmes proportions. Si vous souhaitez l'étirer horizontalement ou verticalement, vous pouvez utiliser les poignées de redimensionnement latérales.

Modifier les paramètres d'habillage du texte

Lorsque vous insérez une image à partir d'un fichier, vous pouvez remarquer qu'il est difficile de la déplacer exactement où vous le souhaitez. Cela est dû au fait que l'habillage du texte pour l'image est défini sur "En ligne avec le texte". Vous devrez modifier le paramètre d'habillage du texte si vous souhaitez déplacer librement l'image ou si vous voulez simplement que le texte s'enroule autour de l'image de manière plus naturelle.

 

Pour faire en sorte que le texte s'enroule autour d'une image :

Sélectionnez l'image autour de laquelle vous souhaitez faire enrouler le texte. L'onglet Format apparaîtra sur le côté droit du Ruban.

Sur l'onglet Format, cliquez sur la commande Habillage du texte dans le groupe Organiser. Ensuite, sélectionnez l'option d'habillage du texte souhaitée. Dans notre exemple, nous sélectionnerons Devant le texte pour pouvoir la déplacer librement sans affecter le texte. Vous pouvez également sélectionner Plus d'options de mise en page pour affiner la mise en page.


Le texte s'enroulera autour de l'image. Vous pouvez maintenant déplacer l'image si vous le souhaitez. Il vous suffit de cliquer dessus et de la faire glisser à l'emplacement souhaité. En la déplaçant, des guides d'alignement apparaîtront pour vous aider à aligner l'image sur la page.

Alternativement, vous pouvez accéder aux options d'habillage du texte en sélectionnant l'image et en cliquant sur le bouton Options de mise en page qui apparaît.

Si les guides d'alignement n'apparaissent pas, sélectionnez l'onglet Mise en page, puis cliquez sur la commande Aligner. Sélectionnez Utiliser les guides d'alignement dans le menu déroulant qui apparaît.

Utiliser un paramètre prédéfini d'habillage du texte

Les paramètres prédéfinis d'habillage du texte vous permettent de déplacer rapidement l'image vers un emplacement spécifique sur la page. Le texte s'enroulera automatiquement autour de l'objet, de manière à rester facile à lire.

Insérer des images en ligne

Si vous n'avez pas l'image que vous souhaitez sur votre ordinateur, vous pouvez trouver une image en ligne à ajouter à votre document. Word propose deux options pour rechercher des images en ligne.

 

OneDrive : Vous pouvez insérer une image stockée sur votre OneDrive. Vous pouvez également associer d'autres comptes en ligne à votre compte Microsoft, tels que Facebook et Flickr.

Recherche d'images Bing : Vous pouvez utiliser cette option pour rechercher des images sur Internet. Par défaut, Bing ne montre que des images sous licence Creative Commons, ce qui signifie que vous pouvez les utiliser pour vos propres projets. Cependant, vous devriez cliquer sur le lien vers le site web de l'image pour voir s'il y a des restrictions quant à son utilisation.

Pour insérer une image en ligne :

Placez le curseur à l'endroit où vous souhaitez que l'image apparaisse.

 

Sélectionnez l'onglet Insérer, puis cliquez sur la commande Images en ligne.

La boîte de dialogue Insérer des images apparaîtra.

Choisissez la recherche d'images Bing ou votre OneDrive. Dans notre exemple, nous utiliserons la recherche d'images Bing.

Appuyez sur la touche Entrée. Les résultats de votre recherche apparaîtront dans la boîte.

 

Sélectionnez l'image souhaitée, puis cliquez sur Insérer.

L'image apparaîtra dans le document.

Lorsque vous ajoutez des images, des vidéos ou de la musique à vos propres projets, il est important de vous assurer que vous avez les droits légaux pour les utiliser. La plupart des choses que vous achetez ou téléchargez en ligne sont protégées par le droit d'auteur, ce qui signifie que vous pourriez ne pas être autorisé à les utiliser. Pour plus d'informations, consultez notre leçon sur le droit d'auteur et l'utilisation équitable.
