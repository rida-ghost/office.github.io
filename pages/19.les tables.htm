Introduction

Une table est une grille de cellules disposées en lignes et en colonnes. Les tables peuvent être utilisées pour organiser tout type de contenu, que vous travailliez avec du texte ou des données numériques. Dans Word, vous pouvez rapidement insérer une table vide ou convertir du texte existant en une table. Vous pouvez également personnaliser votre table en utilisant différents styles et mises en page.

Pour insérer une table vide :

1. Placez le point d'insertion à l'endroit où vous souhaitez que la table apparaisse.

2. Accédez à l'onglet Insertion, puis cliquez sur la commande Table.

3. Cliquez sur la commande Table.

4. Cela ouvrira un menu déroulant contenant une grille. Survolez la grille pour sélectionner le nombre de colonnes et de lignes souhaité.

5. Sélectionnez le nombre de cellules.

6. Cliquez sur la grille pour confirmer votre sélection, et une table apparaîtra.

7. Pour entrer du texte, placez le point d'insertion dans n'importe quelle cellule, puis commencez à taper.

8. Pour naviguer entre les cellules, utilisez la touche Tabulation ou les touches fléchées de votre clavier. Si le point d'insertion se trouve dans la dernière cellule, en appuyant sur la touche Tabulation, une nouvelle ligne sera automatiquement créée.

Pour convertir un texte existant en une table :

1. Dans l'exemple ci-dessous, chaque ligne de texte contient une partie d'une liste de contrôle, comprenant des tâches et des jours de la semaine. Les éléments sont séparés par des tabulations. Word peut convertir ces informations en une table en utilisant les tabulations pour séparer les données en colonnes.

 

2. Sélectionnez le texte que vous souhaitez convertir en table. Si vous utilisez notre fichier d'exercice, vous trouverez ce texte à la page 2 du document.

 

3. Allez dans l'onglet Insertion, puis cliquez sur la commande Table.

4. Sélectionnez Convertir le texte en tableau dans le menu déroulant.

5. Une boîte de dialogue apparaîtra. Choisissez l'une des options sous "Séparer le texte à". C'est ainsi que Word sait ce qu'il doit placer dans chaque colonne.

6. Cliquez sur OK. Le texte apparaîtra sous forme de tableau.

Pour modifier des tableaux :

Vous pouvez facilement changer l'apparence de votre tableau une fois que vous l'avez ajouté à votre document. Il existe plusieurs options de personnalisation, notamment l'ajout de lignes ou de colonnes et le changement du style du tableau.

 

Pour ajouter une ligne ou une colonne :

1.       Passez la souris à l'extérieur du tableau à l'endroit où vous souhaitez ajouter une ligne ou une colonne. Cliquez sur le signe plus qui apparaît.

2.       Une nouvelle ligne ou colonne sera ajoutée au tableau.

3.       Alternativement, vous pouvez faire un clic droit sur le tableau, puis survoler "Insérer" pour voir les différentes options de lignes et de colonnes.

Pour supprimer une ligne ou une colonne :

1. Placez le point d'insertion dans la ligne ou la colonne que vous souhaitez supprimer.

2. Faites un clic droit, puis sélectionnez Supprimer les cellules dans le menu.

4.       Une boîte de dialogue apparaîtra. Choisissez Supprimer la ligne entière ou Supprimer la colonne entière, puis cliquez sur OK.

4. La ligne ou la colonne sera supprimée.

Pour appliquer un style de tableau :

Les styles de tableau vous permettent de changer instantanément l'apparence de votre tableau. Ils contrôlent plusieurs éléments de design, notamment la couleur, les bordures et les polices.

 

1.       Cliquez n'importe où dans votre tableau pour le sélectionner, puis cliquez sur l'onglet "Conception" (Design) situé tout à droite du Ruban (Ribbon).

2.       Repérez le groupe "Styles de tableau", puis cliquez sur la flèche déroulante "Plus" pour voir la liste complète des styles.

3.       Sélectionnez le style de tableau que vous souhaitez.

4.       Le style de tableau sera appliqué à votre tableau.

Pour modifier les options de style de tableau :

Une fois que vous avez choisi un style de tableau, vous pouvez activer ou désactiver différentes options pour changer son apparence. Il existe six options : Ligne d'en-tête, Ligne totale, Lignes alternées, Première colonne, Dernière colonne et Colonnes alternées.

 

1. Cliquez n'importe où dans votre tableau, puis accédez à l'onglet "Conception" (Design).

2. Repérez le groupe "Options de style de tableau", puis cochez ou décochez les options désirées.

3. Le style de tableau sera modifié en conséquence.

4. En fonction du style de tableau que vous avez choisi, certaines options de style de tableau peuvent avoir un effet différent. Vous devrez peut-être expérimenter pour obtenir l'apparence souhaitée.

Pour appliquer des bordures à un tableau :

1.       Sélectionnez les cellules auxquelles vous souhaitez appliquer une bordure.

2.       Utilisez les commandes de l'onglet "Conception" (Design) pour choisir le style de ligne, le poids de la ligne et la couleur du stylo désirés.

3. Cliquez sur la flèche déroulante sous la commande "Bordures".

4. Choisissez un type de bordure dans le menu.

5.       La bordure sera appliquée aux cellules sélectionnées.

Modifications d'un tableau à l'aide de l'onglet Mise en page

In Microsoft Word, the Layout tab provides various controls and options for modifying tables. Here's a brief overview of some common tasks you can perform using the Layout tab:

 

1. **Insert Above/Below**: These buttons allow you to add a new row above or below the currently selected row.

 

2. **Insert Left/Right**: These buttons add a new column to the left or right of the currently selected column.

 

3. **Delete**: Use the Delete button to remove the selected rows, columns, or cells.

 

4. **Merge Cells**: Merge selected cells into a single cell. This can be useful for creating header cells or combining cells for a specific layout.

 

5. **Split Cells**: Split a selected cell into multiple cells horizontally or vertically. This is handy for dividing a cell into smaller sections.

 

6. **Alignment**: Adjust the alignment of text within the selected cells. You can choose from options like left-aligned, center-aligned, or right-aligned text.

 

7. **Distribute Rows/Columns**: These options help you evenly distribute the height of rows or the width of columns, making your table look more organized.

 

8. **Cell Size**: Here, you can set the exact height and width of selected cells or rows, ensuring precise control over your table's layout.

 

9. **Table Properties**: This opens a dialog box where you can configure advanced table settings, such as the table's size, alignment, text wrapping, and more.

 

10. **AutoFit**: Adjust the table's size automatically based on its content or distribute columns evenly.

 

These options on the Layout tab give you a high degree of control over the structure and appearance of your tables in Microsoft Word.
